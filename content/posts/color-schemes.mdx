Nordlys no solo incluye un modo oscuro y claro, sino también esquemas de colores predefinidos que están listos para usar, así como la opción de agregar esquemas personalizados.

## Usando un Esquema Predefinido

Simplemente establece `colorScheme` en la configuración del tema (`theme.config.ts`) a uno de los esquemas predefinidos.

## Agregando un Nuevo Esquema

Además de los esquemas de colores predefinidos, es muy simple agregar un esquema personalizado. Solo elige un color primario (acento) y un color de fondo, y agrégalos a `color-schemes.css`. Asegúrate de usar los valores RGB de los colores en el mismo formato que se muestra.

```css src/style/color-schemes.css
@layer base {
  .scheme-custom {
    --accent: 0, 0, 0;
    --accent-bg: 255, 255, 255;

    &[data-mode='dark'] {
      --accent: 255, 255, 255;
      --accent-bg: 0, 0, 0;
    }
  }
}
```

Si quieres usar el esquema recién definido, configúralo en `theme.config.ts`.

```ts src/theme.config.ts
export default defineThemeConfig({
  colorScheme: 'scheme-custom'
  // ...
})
```

Lo más probable es que tu IDE o editor muestre un TypeError indicando que el esquema no es una opción válida. Si bien esto no es un error crítico, puedes solucionarlo agregando `scheme-custom` como una opción válida al array `ColorSchemes` en la definición de tipos.

```ts src/types.ts
export const ColorSchemes = [
  // otros esquemas
  'scheme-custom'
] as const
```

¡Y eso es todo!
